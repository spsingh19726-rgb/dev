# Rudraksha Agro Products - Demo Site

This is a lightweight Vite + React demo of the Rudraksha Agro Products website.

## Run locally
```bash
npm install
npm run dev
# open http://localhost:5173
```

## Contents
- `src/App.jsx` - main React app with pages
- `src/main.jsx` - entry
- `index.html` - Vite entry
